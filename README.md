# Contacts Lite
Lightweight contacts and phone call application for devices with Android 8.0 (API level 26) and above

## Tech Stack & Libraries
- Room Persistence
- Fragment
- Coroutines
- MVVM architecture
- Lifecycle
- ViewModel
- LiveData
- Splash Screen API
- Country Code Picker Library ([link](https://github.com/hbb20/CountryCodePickerProject))
